To produce the ergoscore related to given recording : 
You need to call python from a power shell terminal (from the folder containing Ergoscore_mainRevamp.py and ErgoScoreToolsRevamp.py).
The file, containing the data, should be located in the same folder (where the .py are) but it is possible to give the path to reach it. 
If the file contain only one data sheet, don't specify any name.
